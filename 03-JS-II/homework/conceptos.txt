- For: Es uno de los terminos reservados en el lenguaje de programación, su significado es "para" y, similar al "if", se emplea como un operador de sentencia condicional para crear un bloque de código mediante el cual se puede controlar el comportamiento de varias operaciones y/o expresiones de manera repetida y decidir el resultado de las mismas según ciertas condiciones preestablecidas. A este tipo de bloques de código se les conoce como bucles, y al creado por el operador "For" se le conoce propiamente como "bucle For". Este bucle en particular se le construye indicando primero la palabra "for",  seguida de un espacio encerrado en parentesis donde se van a especificar tres elementos condicionales, separados uno de otro por un punto y coma: primero se define e inicializa una variable que sirve como contador en la operación a realizar, de segundo define una condición sobre la variable que hemos definido y se fija el número de veces que queremos que se ejecute la tarea y de tercero se especifica una expresión de aumento para la variable; luego del área entre parentesis, le sigue un área encerreda entre llaves donde se especificará la expresión o código que se ejecutara de acuerdo a las condiciones previas. Ejemplo,

for (var i = 0; i < 10; i++) {
	console.log(i); //código a ejecutar
}

En este "bucle for" se define entre parentesis la variable "i" con valor inicial en 0, luego a la variable se le fija la condición de que tomara todos los enteros menores que 10, y por último se indica que la variable irá aumentando su valor de uno en uno a partir de su valor inicial. Entre llaves, el código a ejecutar imprimirá el valor de la variable "i", 10 veces, desde su valor inicial 0 hasta el número 9. 

- &&: Es el operador de comparación lógico que significa "AND" en Inglés o "Y" en Español, y se usa para comparar el valor de veracidad, "True: Verdadero" o "False: Falso", de dos expresiones y obtener un solo valor de veracidad resultante; el cual solo será "True" siempre que el valor de veracidad de cada una de la expresiones comparadas es "True", en cualquier otro caso siempre será "False".

- ||: Es el operador de comparción lógico "OR" en Inglés u "O" en español, y al igual que el operador "&&", se usa para comparar el valor de veracidad de dos expresiones; solo que para este operador, el valor de veracidad resultante será "True" siempre que al menos una de las dos expresiones comparadas tenga valor de veracidad "True" pero si ambas tienen valor "False", el valor resultante será "False".

- !: Es el operador de comparación lógico "NOT" en Inglés y "NO" en Español, y se usa para negar e invertir el valor de veracidad de una expresión; es decir, si una expresión tiene valor de veracidad "True", el operador "NOT" lo cambiara por "False" y viceversa. Además, este operador se puede usar en combinación con los operadores "&&" y "||" para modificar lo valores de veracidad resultantes de las expresiones que se comparen con dichos operadores.
